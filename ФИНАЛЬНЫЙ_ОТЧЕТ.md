## Соответствие стандартам и ограничениям

### ✅ C++ Standard
- Используется C++17 (флаг `-std=c++17`)
- Ключевое слово `mutable` соответствует C++11 и выше
- `= delete` для операций копирования соответствует C++11 и выше

### ✅ Embedded Constraints
- Без динамического выделения памяти (нет `new`, `malloc`)
- Без исключений (флаг `-fno-exceptions`)
- Без RTTI (флаг `-fno-rtti`)
- Стек: ~1,369 байт максимум (33.4% от 4 КБ)
- Используется `std::array` с фиксированным размером
- Используются фиксированные буферы

### ✅ Thread Safety (ChibiOS RTOS)
- `mutable Mutex` для const-корректности
- `MutexLock<LockOrder::DATABASE_MUTEX>` для детерминированного порядка блокировок
- `AtomicFlag` для атомарных флагов

### ✅ RAII
- `FileGuard` предотвращает утечки ресурсов
- Деструктор гарантированно закрывает файл
- Операции копирования явно удалены
